What is RainHub?

RainHub is a Roblox exploit gui that supports multiple games like Murder Mystery 2, The underground war and others coming soon.

When did RainHub start?

If you are going by the creation date of the github repo (J0CKSTRAP/Rainhub/RainHubOP) it was made some time around august 2022 but development on RainHub started in may-july of 2022 as a gui only for the underground wars. RainHub V1 was very limited in what it could do like for TUW it could not auto return to base after using one of the teleport buttons it also had a single step tween teleport system and the anticheat would detect it. I stopped working on updates for RainHub soon after I made the github repo around august 2022. I lost the source code when I moved pcs so I quit working on it completely until middle of ​february 2023 my friend gave me ​motivation to remake RainHub. Also at the time I kind of wanted to remake RainHub with Solaris Lib. So RainHub V2 was born.
